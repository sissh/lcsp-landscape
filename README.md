# LCSP Landscape

Landscape basé sur le modèle du CNCF : https://github.com/cncf/landscape

Généré avec l'outil landscape2 : https://github.com/cncf/landscape2

Pour déployer la page :

`landscape2 build --data-file landscape.yml --settings-file settings.yml --output-dir <DOSSIER_CIBLE> --logos-path hosted_logos/`

`landscape2 serve --landscape-dir <DOSSIER_CIBLE>`
